# 传输助手 (Transfer Assistant)

像聊天一样简单！电脑手机跨端互传，支持局域网直连 + 云端推送双模式，文字文件秒级同步。

---

### **1. 动作信息**
- **动作名称**：传输助手
- **一句话简介**：跨端互传神器，局域网极速直连 + 云端推送保底，免安装、零配置。
- **关键词**：传输助手;文件传输;手机电脑互传;剪贴板同步;跨平台;免安装;局域网;无线U盘

---

### **2. 开发者说：为什么开发这个动作？**
有时我在办公时，总被一些“小事”困扰：
只是想把手机上的一段话或者一张照片传到电脑，却要打开臃肿的社交软件，还要忍受画质压缩和隐私被窥探的风险；想用局域网传输工具，却总是因为防火墙拦截、WiFi 隔离或者 IP 变动导致连接失败，折腾半天还没传好。

我渴望有一个工具：**像微信对话框一样简单，不需要登录账号，且无论网络环境多复杂都能一秒连接。**

于是我开发了《传输助手》。它不仅能利用局域网实现“零流量”极速大文件传输，还集成了 Quicker 原生 Push API。即使局域网不通，云端通道也能确保文字和通知秒级直达。它彻底改变了我的跨端协作流程，让我能全身心地投入到真正重要的工作中。

---

### **3. 核心功能**
- **⚡ 双模驱动**：优先走局域网极速通道（传文件首选）；当局域网受限时自动激活云端推送模式（传文字保底）。
- **💬 IM 交互体验**：对话流式 UI，支持图片预览、视频一键播放、文字点击拷贝。
- **📸 零配置配对**：动态二维码自动携带所有配置参数，扫码即连，无需手动输入 IP 或 Key。
- **🛡️ 隐私与持久化**：本地历史记录自动保存，支持自定义文件保存目录，数据完全由你掌控。
- **📋 剪贴板同步**：手机发送的文字会自动进入电脑剪贴板，实现无感同步。

---

### **4. 使用说明**

#### **第一步：启动动作**
在 PC 端运行动作，会弹出“传输助手 - 电脑工作台”窗口。

#### **第二步：手机配对**
1. **扫码连接**：点击窗口左侧的二维码放大，用手机（建议使用手机自带相机、浏览器、支付宝或 QQ）扫码即可进入 Web 端。
2. **快捷访问**：建议在手机浏览器中将网页“添加到主屏幕”，下次使用只需点击图标，像 App 一样方便。

#### **第三步：开始传输**
- **PC -> 手机**：直接将文件拖入电脑窗口，或点击 `+` 号选择。
- **手机 -> PC**：在 Web 端输入文字或点击附件上传。接收到的文字会自动进入电脑剪贴板。

#### **第四步：进阶配置（强烈推荐）**
为了保证 100% 的连接成功率，建议开启推送模式：
1. 在 PC 端窗口点击 **⚙️ 设置** 图标。
2. 勾选 **“启用推送 API”**。
3. 填入你 Quicker 设置中的 **推送验证码 (Push Key)**（在 Quicker 设置 -> 推送面板中查看）。
4. 开启后，即使手机和电脑不在同一个 WiFi 下（例如手机用 5G），依然可以实现云端文字互传。

---

### **5. 注意事项**
- **防火墙**：如果局域网无法连接，请检查电脑防火墙是否允许了 3001 端口。
- **微信扫码**：由于微信限制，微信扫码后请点击右上角 [...] 选择“在浏览器中打开”以获得最佳体验。
